Política de Privacidade – Bugster Bot

1. Coleta de Dados
O Bugster pode coletar dados limitados como:
- IDs de usuários e mensagens para fins de moderação;
- Dados de economia virtual (saldo, histórico de transações);
- Logs de comandos e interações.

Todos os dados são utilizados exclusivamente para as funções internas do bot e não são compartilhados com terceiros.

2. Segurança
Todos os dados são armazenados com medidas de segurança adequadas. Não há coleta de informações sensíveis ou pessoais além do necessário para a operação do bot.

3. Remoção de Dados
O usuário ou administrador pode solicitar a exclusão de dados específicos a qualquer momento, mediante contato direto com os desenvolvedores do Bugster.

4. Alterações na Política
Esta política pode ser atualizada periodicamente. Quaisquer alterações relevantes serão informadas nos canais oficiais.
